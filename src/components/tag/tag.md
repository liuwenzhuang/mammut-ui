# Tag 标签

## 使用示例

```html
<ui-tag color="#fb8d18" borderColor="#ffd1a0" bgColor="#fff7e6">测试</ui-tag>
```

## ui-tag

### API

| 参数        | 说明                                                              | 类型   | 默认值  |
| ----------- | ----------------------------------------------------------------- | ------ | ------- |
| color       | 文本颜色                                                          | string | #000    |
| borderColor | 边框颜色                                                          | string | #d9d9d9 |
| bgColor     | 背景颜色，如果指定了此属性，但未指定 color，则文本颜色默认为 #fff | string | #fafafa |
