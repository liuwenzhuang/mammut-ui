# Checkbox 复选框组件

## 使用示例

模板
```html
<ui-checkbox checked="{checked}" disabled="{disabled}" on-change="{this.handleChange($event)}"></ui-checkbox>
```

## API
| 参数 | 描述 | 默认值 |
| --- | --- | --- |
| checked | 是否选中 | false |
| disabled | 是否禁用 | false |
