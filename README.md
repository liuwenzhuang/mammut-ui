# MammutUI 猛犸组件系统
> 项目开发首页 http://localhost:8444

## 运行

1. 安装依赖 `npm install`
2. 运行开发服务器 `npm start`
3. [组件示例目录](http://localhost:8444/examples/main-nav)

## 项目结构

```
    ├── .eslintrc
    ├── README.md
    ├── assets - 静态资源文件
    ├── doc - 示例文件夹
    ├── lib - 生成的打包文件
    ├── node_modules
    ├── package-lock.json
    ├── package.json
    ├── scripts - 脚本文件
    └── src - 源文件
```
